Detailed Description

A singleton atlas of widgets.

Atlas is a class that acts as a collection of Interface pointers. It provides methods for inserting, removing, and sorting widgets, as well as accessing the collection itself. Being a Singleton, it ensures there is only one instance of this collection in the program.

This class is called automaticaly by the widget when they are created or when they have their depth edited. This singleton is created inside the WidgetModule.
