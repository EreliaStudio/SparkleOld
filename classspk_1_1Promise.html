Detailed Description

template<typename TType>
class spk::Promise< TType >

Class for providing a promise with a value.

This class provides a promise that can be fulfilled with a value. It allows subscribing to the promise to receive the value when it becomes available.

Template Parameters
TTypeThe type of the value.
