Detailed Description

template<typename TEvent>
class spk::Observer< TEvent >

Observer class template.

This class template represents an observer that can subscribe to events and notify registered callbacks when the events occur.

Template Parameters
TEventThe type of the event.

Notify the observers of an event.

This function notifies all registered callbacks for the specified event. It retrieves the callback container for the event and calls each callback function in the container.
