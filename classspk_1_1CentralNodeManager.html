Detailed Description

A no-graphics widget class for managing a spk::CentralNode.

The CentralNodeManager class is a graphical user interface widget that can manage an instance of spk::CentralNode. This includes rendering the central node status on the GUI, handling geometry changes, and performing periodic updates.
