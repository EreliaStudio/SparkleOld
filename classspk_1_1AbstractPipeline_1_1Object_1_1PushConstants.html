Detailed Description

A class to handle Push Constants within a shader layout.

The PushConstants class serves as an extension of the spk::FieldMap. It is designed to work with push constants in the context of a shader layout, allowing for more streamlined management of these constants.

Note
This class is derived from spk::FieldMap.
