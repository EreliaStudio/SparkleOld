Detailed Description

template<typename TState>
class spk::StatedObject< TState >

A class who manage state switching. This class representing an object who can be in different state. the class additionnaly store lambdas to call uppon specific state. Each lambda is provided to the user as Contract object.

Template Parameters
TStateThe status class/structure/enum/type in which the object can be set
