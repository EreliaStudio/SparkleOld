Detailed Description

template<typename TType, typename = std::enable_if_t<std::is_standard_layout<TType>::value>>
class spk::StandardType< TType, typename >

A class template for standard layout types, derived from Memento.

This class is used to save and load the state of a standard layout type.

Template Parameters
TTypeThe type of the standard layout type.
