\doxysubsection{Detailed Description}
A concrete class derived from \mbox{\hyperlink{classspk_1_1IInput}{IInput}} that represents keyboard input. 

\mbox{\hyperlink{classspk_1_1KeyInput}{Key\+Input}} is a specific implementation of \mbox{\hyperlink{classspk_1_1IInput}{IInput}} for keyboard inputs. It encapsulates a keyboard key and the expected status of the key. The encapsulated function is executed if the current status of the specified keyboard key matches the expected status.
