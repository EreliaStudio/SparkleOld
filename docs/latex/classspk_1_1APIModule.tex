\doxysubsection{Detailed Description}
Class for managing the API interactions. 

This class handles the API interactions and updates for the window, mouse and keyboard. It uses thread-\/safe queues to ensure safe multi-\/threaded operations.

This class handles the API interactions and updates for system, window, mouse, and keyboard. It uses a pool of \mbox{\hyperlink{classspk_1_1DataBuffer}{Data\+Buffer}} as a singleton instance and thread-\/safe queues to ensure safe multi-\/threaded operations.
