\doxysubsection{Detailed Description}
Abstract class providing a common interface for input classes in the \textquotesingle{}spk\textquotesingle{} library. 

An \mbox{\hyperlink{classspk_1_1IInput}{IInput}} object encapsulates a function that may need to be executed at a regular interval. The function is executed if the {\ttfamily \mbox{\hyperlink{classspk_1_1IInput_af79e6745cbed467b0ba91ce1ff741662}{\+\_\+is\+Execution\+Needed()}}} method returns true. The {\ttfamily \mbox{\hyperlink{classspk_1_1IInput_a92ab93dccf24b9c0809cd141a4f975d2}{\+\_\+skip\+Condition()}}} method can be overridden to provide a condition under which the update is skipped.
