\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TType$>$\newline
class spk\+::\+Pool$<$ TType $>$\+::\+Object}

An internal class to manage objects retrieved from the pool. 

This class is designed to keep track of the memory it manages and to automatically return it to the pool when it is no longer needed.
