\doxysubsection{Detailed Description}
A concrete class derived from \mbox{\hyperlink{classspk_1_1IInput}{IInput}} that represents mouse input. 

\mbox{\hyperlink{classspk_1_1MouseInput}{Mouse\+Input}} is a specific implementation of \mbox{\hyperlink{classspk_1_1IInput}{IInput}} for mouse inputs. It encapsulates a mouse button and the expected status of the button. The encapsulated function is executed if the current status of the specified mouse button matches the expected status.
