\doxysubsection{Detailed Description}
A class to profile and manage the frames-\/per-\/second rates of the application. 

This class is responsible for tracking the number of frames and updates that the application processes every second (known as the \char`\"{}render IPS\char`\"{} and \char`\"{}update IPS\char`\"{} respectively).
