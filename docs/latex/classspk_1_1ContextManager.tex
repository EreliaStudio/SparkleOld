\subsubsection*{template$<$typename TContext\+Type$>$\newline
class spk\+::\+Context\+Manager$<$ TContext\+Type $>$}

\mbox{\hyperlink{classspk_1_1ContextManager}{Context\+Manager}} is a singleton template class that manages Read\+Write and Read\+Only access to a common context. 

This class provides access to a context object for read and write operations in a thread safe manner. It uses double buffering technique, thus ensuring that read operations are not blocked during write operations.
