\doxysubsection{Detailed Description}
A singleton class that handles profiling using various chronometers. 

This class is responsible for managing multiple chronometers, allowing the software to keep track of time metrics for profiling purposes. As a singleton, only one instance of this class can exist. This ensures consistency across the entire software.
