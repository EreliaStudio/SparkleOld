\doxysubsection{Detailed Description}
A concrete class derived from \mbox{\hyperlink{classspk_1_1IInput}{IInput}} that represents mouse movement input. 

\mbox{\hyperlink{classspk_1_1MouseMovementInput}{Mouse\+Movement\+Input}} is a specific implementation of \mbox{\hyperlink{classspk_1_1IInput}{IInput}} for mouse movements. The encapsulated function is always executed in every update, because the {\ttfamily \+\_\+is\+Execution\+Needed()} method always returns true.
