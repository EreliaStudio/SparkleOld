\doxysubsection{Detailed Description}
Abstract base class for widgets. 

This class defines the interface and common functionality for widgets. It inherits from \mbox{\hyperlink{classspk_1_1InherenceObject}{Inherence\+Object}} and \mbox{\hyperlink{classspk_1_1ActivableObject}{Activable\+Object}}.
