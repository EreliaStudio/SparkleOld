\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename TType$>$\newline
class spk\+::\+Value$<$ TType $>$\+::\+Default}

The inner class for managing the default state of the value. 

This inner class represents the default state of the value. It holds the actual value, keeps track of subscribers, and triggers edition callbacks when the value is changed.
