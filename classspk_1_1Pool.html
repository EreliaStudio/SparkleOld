Detailed Description

template<typename TType>
class spk::Pool< TType >

A generic pool class that manages the creation and deletion of objects.

This class is designed to handle memory in an efficient manner by reusing previously allocated objects. This avoids frequent memory allocation and deallocation, which can be costly for performance.

The Pool class provides a mechanism for 'reserving' a number of objects of a specific type, and then 'obtaining' those objects for use. When an object is no longer needed, it is returned to the pool to be reused later.

Template Parameters
TTypeThe type of objects that the pool will manage.
