Detailed Description

template<typename TType>
class spk::Pool< TType >

A template class for object pooling.

This class implements an object pool pattern. It is useful for managing a collection of reusable objects of type TType. Object pools can improve performance in situations where the cost of initializing an instance of the class is high, the rate of instantiation of a class is high, and the number of instantiations in use at any one time is low.
