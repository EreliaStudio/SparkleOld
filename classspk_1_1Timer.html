Detailed Description

Class for creating and managing a timer.

This class allows you to create a timer with a specified duration. The timer can be started, stopped, and checked for its status.

Member Enumeration Documentation

◆ State

enum spk::Timer::State
strong

An enum class to represent the state of the timer.

This enum class has three states: Idle, Running, and Timeout. The timer is in the Idle state by default, changes to Running when it starts, and goes to Timeout when the duration has passed.

Enumerator
Idle 

Timer is idle (not running and not timed out).

Running 

Timer is currently running.

Timeout 

Timer has timed out.
