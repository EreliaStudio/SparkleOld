Detailed Description

template<typename TType>
class spk::Singleton< TType >

Singleton class template.

This class template provides a base implementation for creating singleton objects. It ensures that only one instance of the object is created and provides methods to access and release the instance.

Template Parameters
TTypeThe type of the singleton object.
