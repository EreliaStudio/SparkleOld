Detailed Description

The Socket class is a wrapper around a network socket.

The Socket class encapsulates a network socket, and includes methods for establishing connections, sending messages, and receiving messages.

The Socket class is a higher-level representation of a network socket, which includes methods for establishing connections, sending messages, and receiving messages.

The Socket class extends from the Object class. The Server, Client, and Acceptor classes are friends of Socket, giving them access to its private members and methods.
