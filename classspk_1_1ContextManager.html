Detailed Description

template<typename TContextType>
class spk::ContextManager< TContextType >

ContextManager is a singleton template class that manages ReadWrite and ReadOnly access to a common context.

This class provides access to a context object for read and write operations in a thread safe manner. It uses double buffering technique, thus ensuring that read operations are not blocked during write operations.

Template Parameters
TContextTypeThe type of the context object this class is managing.
