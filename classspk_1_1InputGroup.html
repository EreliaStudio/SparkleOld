Detailed Description

Manages a group of input objects.

An InputGroup allows the grouping and management of multiple input objects, including adding, removing, and updating inputs collectively.

Adds an Input object to this group.

The given Input object will be added to the internal list, allowing it to be managed collectively with other inputs in the group.

Removes an Input object from this group.
