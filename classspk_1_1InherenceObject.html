Detailed Description

template<typename TType>
class spk::InherenceObject< TType >

InherenceObject class template.

This class template represents an object that can be part of an inheritance hierarchy. It allows setting a parent object, adding child objects, and defining birth and death callbacks.

TTypeThe type of the object.
