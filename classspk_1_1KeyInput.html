Detailed Description

A concrete class derived from Input that represents keyboard input.

KeyInput is a specific implementation of Input for keyboard inputs. It encapsulates a keyboard key and the expected status of the key. The encapsulated function is executed if the current status of the specified keyboard key matches the expected status.
